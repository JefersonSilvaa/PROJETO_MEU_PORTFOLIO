Guia da pasta de conteúdos

Esta página organiza o que existe em Estudo-FRONT-END para você visualizar rápido: o que estudar primeiro, onde estão os exercícios e quais pastas usar para prática de portfólio.

Trilhas de estudo recomendadas

Sugestão de ordem para ter uma evolução consistente e sem pular fundamentos.

  1. 1. Fundamentos de estrutura (HTML + CSS básico)

    Aprenda marcação semântica e estilos essenciais para montar páginas limpas e organizadas.

  2. 2. Lógica de programação

    Fortaleça pensamento algorítmico com exercícios antes de avançar no JavaScript mais profundo.

  3. 3. JavaScript básico e DOM

    Entenda variáveis, funções, condicionais e manipulação de elementos da página.

  4. 4. HTML/CSS intermediário

    Avance para flex, grid, animações e recursos visuais para interfaces mais profissionais.

  5. 5. Projetos de portfólio

    Aplique tudo em projetos reais para consolidar habilidades e montar vitrine de trabalhos.

Acervo detalhado por pasta

Clique em cada bloco para ver explicação, foco de estudo e acesso direto à pasta.

HTML5 + CSS3 (Básico)

Contém exercícios de introdução (ex001 até ex026), incluindo semântica, links, tabelas, formulários e media queries.

  • Objetivo: dominar estrutura e estilização inicial.
  • Quando estudar: início da jornada.
  • Pasta: Abrir no visualizador (HTML5+CSS3 básico)
HTML5 + CSS3 (Intermediário)

Trilha com aulas práticas sobre object-fit, flex, grid, background, SVG, gradientes, keyframes, pseudo-classes e responsividade.

  • Objetivo: evoluir layout e acabamento visual.
  • Quando estudar: após base sólida de HTML/CSS.
  • Pasta: Abrir no visualizador (HTML5+CSS3 intermediário)
JavaScript (Básico)

Conteúdo de introdução até funções, com foco em eventos DOM, operadores, condicionais e repetições.

  • Objetivo: aprender lógica e interação no navegador.
  • Quando estudar: junto da base HTML/CSS.
  • Pasta: Abrir no visualizador (JavaScript básico)
JavaScript (Básico + Intermediário)

Módulos avançando em tipos de dados, loops, objetos, arrays, escopo, this e manipulação detalhada do DOM.

  • Objetivo: ampliar repertório para projetos mais robustos.
  • Quando estudar: após concluir a trilha básica.
  • Pasta: Abrir no visualizador (JavaScript básico + intermediário)
Lógica de Programação

Material com algoritmos e Visualg para desenvolver raciocínio lógico, estruturação e resolução de problemas.

  • Objetivo: fortalecer base antes de desafios mais complexos.
  • Quando estudar: em paralelo ao JavaScript inicial.
  • Pasta: Abrir no visualizador (Lógica de programação)
Projetos para Portfólio

Conjunto de projetos para treino real, como login, contador, portfolio e layouts temáticos.

  • Objetivo: aplicar conteúdo e criar evidências práticas.
  • Quando estudar: após fundamentos + JS básico.
  • Pasta: Abrir no visualizador (Projetos de portfólio)
Aplicativos (VS Code, Git e GitHub)

Instaladores de ferramentas essenciais para desenvolvimento e versionamento.

  • Objetivo: preparar ambiente de estudo e trabalho.
  • Quando estudar: antes de iniciar projetos versionados.
  • Pasta: Abrir no visualizador (Aplicativos)

Como usar esta organização no dia a dia

Plano semanal simples

Segunda a quarta: aulas teóricas e exercícios. Quinta: revisão e refatoração. Sexta: mini projeto aplicado.

Regra de progressão

Só avance para o próximo módulo quando conseguir reproduzir o anterior sem copiar código pronto.

Portfólio contínuo

A cada 2 módulos concluídos, publique um projeto novo para registrar evolução técnica e visual.
