# Task Manager - Versao Simples (Admin)

Sistema organizado para uso direto no frontend, sem depender de login ou backend para funcionar.

## Como funciona

- Ao abrir [frontend/index.html](frontend/index.html), entra automaticamente no painel principal.
- O painel principal esta em [frontend/dashboard.html](frontend/dashboard.html).
- O admin pode criar e enviar atividades para Joao, Maria e Pedro.
- A lista vem com atividades de teste.
- Tudo fica salvo localmente no navegador para testes rapidos.

## Funcionalidades prontas

- Tela Admin principal.
- Adicionar atividade com responsavel e data.
- Editar atividade.
- Excluir atividade.
- Marcar atividade como concluida.
- Filtro: todas, pendentes, concluidas.
- Botao "Reiniciar Lista" para voltar ao estado inicial.
